## Compilation

### [UnityCompileTools](https://github.com/CrossQuestBS/UnityCompileTools)
Unity tools and additional files required for compiling.

### [BeatSaberDependencies](https://github.com/CrossQuestBS/BeatSaberDependencies)
Beat Saber specific dependents required for compiling.

## Mods

### [BeatSaverDownloader](https://github.com/CrossQuestBS/BeatSaverDownloader)
Mod for Beat Saber to download songs from BeatSaver.com in-game 

### [SongCore](https://github.com/CrossQuestBS/SongCore)
A plugin for handling custom song additions in Beat Saber. 

### [ImageCoverExpander](https://github.com/CrossQuestBS/ImageCoverExpander_)
Beat Saber mod that expands the artwork cover of a song. 

### [BeatSaberQuestPatch](https://github.com/CrossQuestBS/BeatSaberQuestPatch)
Patches PC Beat Saber to use Quest related code

## Libraries

### [SiraUtil](https://github.com/CrossQuestBS/SiraUtil)
A powerful utility mod which expands the capabilities and provides more tools to Beat Saber modders. 

### [Accord](https://github.com/CrossQuestBS/Accord)
C# Ahead-of-time patcher with runtime handling of patches.

### [BSIPA](https://github.com/CrossQuestBS/BeatSaber-IPA-Reloaded)
Fork of BSIPA that does things Ahead-of-time (AOT) instead of doing things during runtime.

### [BeatSaberMarkupLanguage](https://github.com/CrossQuestBS/BeatSaberMarkupLanguage)
An easy way to set up your UI in Beat Saber without manually creating new objects and parenting them.

### [Beat-Saber-Utils](https://github.com/CrossQuestBS/Beat-Saber-Utils)
A basic library for mods to use, does nothing on its own
