## Prerequisite

You need to have the following installed
* Java JDK
* .NET Runtime 10 (?)

## Installation

1. Download the standalone version for your platform on CrossQuest [Github Releases](https://github.com/CrossQuestBS/CrossQuest/releases)
2. Extract the zip folder to place such as User folder
3. Add path to the CrossQuest CLI into `PATH` on Windows or `$PATH` on Unix (See guides online for more)
4. Now you are done and can follow to setup
